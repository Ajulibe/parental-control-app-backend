SETTING UP REDIS
to start up the redis server use the command: redis-server
to access the gui for redis use the command: redis-commander

TO SEE ALL JOBS
to access bull dashboard go to: http://localhost:3000/queues


STARTING UP MONGODB
to start the mongo community use - brew services start mongodb-community@6.0
to check if the service is running - brew services list
to start mongo db use the command: mongosh
